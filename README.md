# Inbox Concierge — Frontend

React frontend for the Inbox Concierge email classification app.

## Quick Start

```bash
npm install
cp .env.example .env  # Set VITE_API_URL
npm run dev
```

See the backend repo's [SETUP.md](https://github.com/eliplutchok/inbox-concierge-backend/blob/main/SETUP.md) for full setup instructions.
